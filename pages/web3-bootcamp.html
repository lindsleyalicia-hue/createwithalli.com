🔐 Dev OPSEC Rules

Non-negotiable. Learn these before you touch a line of Solidity.

🚫

NEVER use your main wallet for development. Create a dedicated dev wallet — fund only what you need for gas.

🚫

Store private keys in .env — NEVER in code or Notion. .env goes in .gitignore before the first commit.

✓

Use Anvil for local testing before any testnet deployment.

✓

Audit every contract with Slither before mainnet — no exceptions.

✓

Keep dev wallet and cold wallet completely separate — different seed phrases, different devices.

✓

Document your deployed contract addresses — don't lose them.
